# TimeTracker Tasks - Трекер задач и времени

## Описание
WPF приложение для управления задачами с учетом времени по методологии MVVM.

## Технологии
- .NET 6.0
- WPF
- Entity Framework Core
- SQLite
- xUnit + Moq
- MVVM Architecture

## Сборка и запуск

### Требования
- Visual Studio 2022
- .NET 6.0 SDK

### Инструкция по запуску

1. **Откройте решение** `TimeTrackerTasks.sln`

2. **Восстановите NuGet пакеты:**
   - ПКМ на решение Восстановить пакеты NuGet

3. **Выполните миграции БД:**
   - Откройте "Консоль диспетчера пакетов"
   - Выберите проект: `TimeTrackerTasks.Data`
   - Выполните команды:
     ```
     Add-Migration InitialCreate
     Update-Database
     ```

4. **Назначьте запускаемый проект:**
   - ПКМ на проект `TimeTrackerTasks.WPF` Назначить запускаемым проектом

5. **Запустите приложение:** F5

### База данных
- **Тип БД:** SQLite
- **Файл БД:** `TimeTrackerTasks.db`
- **Миграции:** Требуется выполнить один раз при первом запуске

### Тестирование
1. **Откройте Test Explorer:** Тест ? Обозреватель тестов
2. **Запустите тесты:** "Запустить все"

## Функциональность
- Создание, редактирование, удаление задач
- Фильтрация задач (Все/Активные/Выполненные)
- Поиск по названию задачи
- Учет времени выполнения задач
- Валидация данных
